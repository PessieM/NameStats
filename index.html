This website uses three different APIs to predict your gender, age and nationality, based on your first name!

  • Gender: The Genderize API allows you to check the statistical probability of a name being male or female.
  • Age: The Agify API allows you to estimate the age of a person based on their first name.
  • Nationality: The Nationalize API allows you to check the statistical probability of a name belonging to a certain nationality.
